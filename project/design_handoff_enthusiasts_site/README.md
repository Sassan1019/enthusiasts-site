# Handoff: ENTHUSIASTS ブランド・ポートフォリオサイト

## Overview
「ENTHUSIASTS」——才能を孤独にしないためのプロデュース共同体——のブランド・ポートフォリオサイト。商品販売や応募獲得を目的としたLPではなく、思想・ビジョン・プロジェクト群・メンバーを静かに提示するための1ページ構成サイト＋独立したプライバシーポリシーページの計2ページ。ダークトーン、知的で余白の美しい研究/思想系ブランドサイトのトーン。

## About the Design Files
このフォルダ内の `.dc.html` ファイルは、社内デザインツール（Claude製のDesign Component形式）で作成した**デザインリファレンス**です。`<x-dc>` や `{{ }}` テンプレート構文など、このツール固有の記法を含むため、**そのままではブラウザ/Vercelで動作しません**。実装時は、このHTMLで表現されている見た目・レイアウト・タイポグラフィ・モーション・インタラクションを、Next.js（推奨・Vercelとの相性が良い）または素の静的HTML/CSS/JSとして**忠実に再構築**してください。まっさらな状態からの新規プロジェクトなので、フレームワーク選定も自由です。

## Fidelity
**High-fidelity（hifi）**。色・タイポグラフィ・余白・アニメーションはすべて最終決定版です。ピクセルパーフェクトに再現してください。

## Interactions & Behavior
- **スクロールリビール**: 各セクションの主要要素に `data-reveal` 相当のフェード+上昇アニメーション。IntersectionObserverで画面下から8%の位置に入ったら発火、`opacity:0→1` / `translateY(28px)→0`、easing `cubic-bezier(0.19,1,0.22,1)`、duration 1.15s。同一セクション内で複数要素がある場合は `data-rd`（0,1,2…）に応じて0.12s刻みでずらす。
- **スクロールプログレスバー**: `scroll` イベント（passive）でパーセンテージを計算し、上部固定バーの`width`を更新。
- **ヒーローのCanvasアニメーション**: 疑似3D球体をアスキー文字（` ;:-=+xX80@#`）で描画するCanvas 2D演出。3Dノイズ（Perlin風のhash-based fbm）で球の陰影パターンを生成し、時間経過で回転・ドリフト。文字色は輝度に応じて4段階のパレット（濃い青→薄い青→白）を割り当て。60fps上限、画面外（IntersectionObserver）ではループを止めて負荷軽減。**開発が難しければ、静的なヒーロー背景画像や軽量なCSSアニメーションに置き換えても許容範囲**（体験の核はタイポグラフィと余白なので必須要件ではない）。
- **ノイズ粒子オーバーレイ**: 画面全体に薄い（opacity 0.04）SVGフラクタルノイズを固定表示し、質感を出している（`grain`トグルでON/OFF可能だったUI上のプロパティ）。
- **ナビのレスポンシブ折り畳み**: 760px以下でセクションリンクを非表示、ロゴ＋CONTACTのみ表示。
- **VISION/PROJECTSのレイアウト切替**: CSS Container Query（`container-type: inline-size`）で各ブロック自身の幅に応じて1カラム⇄2カラムを切り替える。ビューポート基準のブレークポイントではなく、コンテナ基準なので、将来レイアウトを入れ子にしても正しく機能する。
- **画像ホバー**: VISION画像は`scale(1.03)`、MEMBER写真は グレースケール解除+`scale(1.025)`。どちらも1.4s前後のゆったりしたeasing。

## State Management
- コンタクトフォーム: `name, email, company, message, agreed, sent` の各フィールド。送信時に `mailto:` URLを組み立てて `window.location.href` に代入、`sent`をtrueにして完了メッセージを表示。
- スクロール位置に応じたプログレスバーの幅（派生値、状態というよりリアルタイム計算）。
- リビールアニメーションの発火状態（IntersectionObserverが一度発火したら要素ごとに終了、再度隠れても戻さない = 1回限りの演出）。

**Colors**
- 背景: `#08090D`（メイン、ほぼ全面）
- テキスト（明）: `#E8EAEF`
- テキスト（本文・やや抑えめ）: `rgba(232,234,239,0.6〜0.75)` 程度（本文は概ね62〜68%）
- テキスト（ラベル・キャプション）: `rgba(232,234,239,0.35〜0.55)`
- アクセント（濃い青）: `#5C82BE`
- アクセント（薄い青）: `#9FC2E8`
- パネル背景（使う場合）: `linear-gradient(180deg, rgba(19,22,30,0.55), rgba(13,15,21,0.55))`
- ロゴプレート背景: `#FFFFFF`（PROJECTSセクションのみ、意図的な純白）
- ヘアライン: `rgba(232,234,239,0.08〜0.12)`

**Typography**
- 見出し・ラベル・数字: `Archivo`（400/500/600/700/800/900）
- 本文・日本語: `Noto Sans JP`（300/400/500/600/700）
- 英大文字見出し（Hero, セクションH2）は `text-transform: uppercase` + 負のletter-spacing (`-0.005em`〜`-0.015em`)
- ラベル系（セクション番号・英語ラベル・役職）は広いletter-spacing（`0.18em`〜`0.36em`）、10〜11.5px
- 本文行間は広め（line-height 2.1〜2.5）が基調 — 「静かな余白」を作る最重要パラメータ

**Spacing**
- セクション縦パディング: `clamp(120px, 15vw, 200px)`前後（HEROやCONTACTはさらに大きい）
- セクション内の要素間の大余白: 最大 96〜190px（VISION/PROJECTSのブロック間が最も広い）
- コンテンツ最大幅: 1180px（本文中心セクションは880〜960px、CONTACTは1180px）

**Motion**
- Easing: `cubic-bezier(0.19,1,0.22,1)`（ゆったり減速、"quiet strength"の質感の核）
- Duration: リビール 1.15s、画像ホバー 0.9〜1.4s、ボタン/入力 0.3〜0.35s

**Border radius / Shadows**
- 角丸: 使用なし（全体を通して直角のみ — シャープでエディトリアルな印象を意図）
- シャドウ: ほぼ使用なし。奥行きは背景のradial-gradientの光暈のみで表現。

## デプロイに関する補足（Vercel + GitHub想定）
- 新規GitHubリポジトリを作成し、Next.js（推奨）または静的HTML一式をpush → Vercelでそのリポジトリをインポートすれば、以降は `main` ブランチへのpushで自動的に本番へ反映される「継続的デプロイ」になります。
- Xserverで取得済みの独自ドメインは、Vercel側の Project Settings → Domains でドメインを追加し、表示される指示に従ってXserverのDNS設定（AレコードまたはCNAME）を向けるだけで接続できます。
- お問い合わせフォームをサーバーで受信したい場合は、Next.jsのAPI Route（またはVercel Functions）+ メール送信APIの追加実装を検討してください（現状のmailto方式は簡易的な代替です）。
